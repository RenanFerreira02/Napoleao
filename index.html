Conheça o nosso robô, suas peças e como ele pode ajudar no desenvolvimento do seu filho

Energia:

O robô funciona com 6 pilhas, essas pilhas dão a energia necessária para que o Arduino funcione corretamente com os motores e armas.

Arduino:

Arduino central do robô juntamente com o driver do motor para controlas as rodas laterais do robô.

Módulo Bluetooth:

Módulo responsável por criar a conexão como aplicativo que permite o controle do robô.

Rodas:

Rodas laterais com motores que vão para frente, trás, direita e esquerda

Arduino Mega 2560

54 pinos de entradas/saídas digitais, 16 entradas analógicas,4 UARTs (portas seriais de hardware), um oscilador de cristal de 16 MHz.

Motor Shield L293D

Esse chip tem a capacidade de controlar motores DC, servo motores e motores de passo. Ele é chamado de Shield pois pode ser facilmente conectado a todos os pinos do Arduino, por possuir basicamente a mesma carcaça.

Módulo Bluetooth HC-06

O Módulo Bluetooth permite a comunicação entre o arduino e outros dispositivos bluetooth por meio de transferência de dados Serial TTL. Com esse módulo é possível controlar servos, motores, Leds e muito mais!

Servo Motor Metal Gear MG995

Este servo padrão de alto torque pode girar 180 graus (Rotação contínua). Você pode usar qualquer código servo, hardware ou biblioteca para controlar este servo.

Dúvidas frequentes

O robô educacional é uma ferramenta inovadora de aprendizado que combina robótica e programação para ensinar conceitos de ciência, tecnologia, engenharia e matemática (STEM). Ele é projetado para ajudar estudantes de todas as idades a desenvolver habilidades técnicas e criativas de forma divertida e interativa.

Nosso robô educacional é indicado para crianças a partir de [idade mínima recomendada], mas é uma ferramenta que pode ser utilizada por pessoas de todas as idades, incluindo adolescentes e adultos interessados em aprender programação e robótica.

Ao utilizar o robô, os estudantes podem desenvolver habilidades em programação, resolução de problemas, pensamento lógico, criatividade e trabalho em equipe. Além disso, é uma excelente maneira de introduzir conceitos de eletrônica e mecânica de forma prática.

Não, o robô foi desenvolvido para iniciantes. Ele vem com um guia de instruções passo a passo, e o software de programação é intuitivo e fácil de usar. Para usuários mais avançados, também oferecemos opções de programação mais complexas.

O robô utiliza Arduino IDE para programação. Ele é compatível com sistemas operacionais como Windows, MacOS e Linux, e pode ser baixado gratuitamente em nosso site. Também oferecemos uma versão do software para dispositivos móveis Android e iOS.

Sim, oferecemos suporte técnico via e-mail e chat online para ajudar com dúvidas sobre montagem, programação e problemas técnicos que você possa encontrar ao usar o robô.
